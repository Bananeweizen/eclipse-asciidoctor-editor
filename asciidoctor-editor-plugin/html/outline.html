Outline

How to show

If not already active please open Window->Show View->Outline.

What is contained?

You will find there all includes and headlines of current document shown - if nothing is available a message will be shown instead of tree.

How can I go to a headline shown in outline?

You must double click on a tree item

How can I open an include shown in outline?

You must double click on a tree item

When is the tree updated?

Changes on your document are only visible inside outline view when you save the document. Unsaved changes will not appear.
