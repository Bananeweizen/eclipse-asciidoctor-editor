Overview

AsciiDoctor editor is an asciidoctor editor plugin for eclipse

Features

  • WYSWYG editor with view
  • Outline view
  • Quick-Outline dialog
  • Code assistance
  • Syntax highlighting
  • Colors can be customized. Also additional defaults for dark theme exists.
  • Block commenting (CTRL+7)
